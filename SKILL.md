# Keyword News Brief

Use this skill when the user asks for recent news about a keyword or topic.

- Fetch current results. Do not answer from memory.
- Prefer high-factuality reporting such as Reuters, AP, BBC, NPR, PBS, and AFP.
- Use RSS or similarly structured sources when possible.
- Return only the 5 most relevant recent headlines with links.
- Format each item as `- headline | link`
- No explanation or extra text.
